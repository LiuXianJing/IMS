# Description
- IMS is Integrated management system.
- IMS is a project that separates the front and back ends.
- frontend: React.
- backend: Python.

# Run Proejct
## frontend
```
pnpm i
pnpm run dev
```
## bcakend
```
pip install flask
pip install flask-mysqldb
pip install -U flask-cors

flask run
```
